Firmware Update Instructions

Note: These instructions apply to devices running firmware version 1.0.1 or later. You can check your firmware version in Settings > About.

If your device is running version 1.0 (displayed in Settings > About), it requires a different update procedure — please message an administrator on LeShuffler community for assistance.

What You Need

  • PC running Windows 11
  • USB-C to USB-C cable (included with LeShuffler)
    — or USB-C to USB-A if your computer does not have USB-C

Step 1 — Download

Download the update package:

Extract the zip file. It contains two files:

  • LeShuffler_Updater.exe — the update tool
  • LeShuffler.sfu — the firmware file

Step 2 — Connect

Connect LeShuffler to the computer using the USB cable.

Step 3 — Enter Update Mode

With the device OFF, hold both the ✓ (Enter) and ✗ (Escape) buttons simultaneously, then turn on the power switch. Keep holding for about 2 seconds.

On devices with bootloader version 3.1 or later, an update screen appears on the display. On earlier bootloader versions, the screen remains black.

You will hear 3 short beeps followed by 1 long beep confirming update mode.

You can check your bootloader version in Settings > About.

Step 4 — Run the Updater

On your computer, run LeShuffler_Updater.exe.

The updater will detect LeShuffler and the firmware file automatically. Follow the on-screen prompts to start the update (press ENTER when asked for confirmation).

Step 5 — Wait for Completion

Do not disconnect the USB cable during the update.

On devices with bootloader version 3.1 or later, the update progress is shown on the display. On earlier bootloader versions, the screen remains black during the update.

When complete, LeShuffler restarts automatically with the new firmware.

Troubleshooting

“Device not detected”

  • Make sure LeShuffler is in update mode (Step 3) before running the updater.
  • Try a different USB port on your computer.
  • Try a different USB cable.
  • If using USB-C to USB-A, try USB-C to USB-C instead, or vice versa.

“Update failed” or transfer interrupted

  • LeShuffler is safe — the previous firmware is preserved until a new update completes successfully.
  • Power off the device, then repeat from Step 3.

Device does not restart after update

  • Turn the power switch off and back on.
  • The new firmware version is shown in Settings > About.
